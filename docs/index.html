Every control starts on auto — the seed decides. Moving a control pins it; “auto” un-pins it. Changes land while the music plays, without restarting.

Send feedback

This project improves through listening feedback. Describe what you heard (what worked, what didn’t — timestamps help), then download the file and send it to Tom: it captures the exact seed and settings, so the same piece can be reproduced.
