## The changes

1. **Dropdowns show italic “choose…” on auto.** Selects used to display a concrete value (e.g. Length “short”) even when the control was on auto, misrepresenting what the seed would pick. Now every enum control renders an italic, muted **“choose…”** placeholder while unpinned; picking a real option pins it (and selecting “choose…” again returns to auto). Implemented in `app.js` (`controlRow` adds a `value=""` placeholder option; `syncCtl` toggles a `.placeheld` class + the select value by pin state; `refreshDynamicControls` does the same for the dynamic palette dropdown) + a `.ctl select.placeheld { font-style: italic; color: var(--muted) }` rule.
2. **New piece + Continuous moved beside Play.** Dropped `margin-left: auto` from `.transportPrimary` so they sit just right of the big Play button instead of flush-right.
3. **Operating instructions moved to About.** Removed the inline auto/pin `autoNote` from the player and the “check to add / uncheck to remove” instruments hint from Advanced; rewrote them (plus tiers, New piece/Continuous/Reset/Copy-link/Background) into three new About sections — **Using the controls**, **Choosing the instruments**, **The seed** — with the instruments text expanded to explain that **the number and choice of instruments is decided at composition time** (the composer draws an ensemble of roles from style + energy/density + seed; unchecking reliably removes, but checking only sounds if the piece has a role for that voice).
4. **Send Feedback removed from the player.** Deleted the `<details class="fb">` block; the feedback code stays in `app.js` (the `#fbDownload` listener is now guarded so its absence is fine) — the capability is retained but no longer surfaced.
5. **Version → v1.0.0.** `SITE_VERSION` in `app.js`. Added a versioning convention to `CLAUDE.md` (patch = tweaks/fixes, minor = new features, major = overhauls; bump on user-visible changes, note it in the session log).
6. **Seed tooltip + About explanation.** Added a `title=` mouseover on the seed label (brief role of the seed) and a longer **The seed** section on About.
7. **Reset relocated to the AUTO column.** Moved `#reset` out of the transport row into a right-aligned row at the top of the controls section, styled to match the AUTO chips (same 3.6rem width, same amber border, aligned above the Energy row’s chip); dropped the `⟲` glyph. Also adds welcome space between the style rail and Energy.
8. **Docs sync.** Brought `CLAUDE.md`, `continue-prompt.md`, and `README.md` current: the site is built **and live at v1.0.0** with the Console design + a standalone About page; `docs/css/`, `docs/fonts/`, `docs/about.html` noted in the repo map; the improvement loop is the ongoing track.
9. **De-duplicated the status line.** The now-playing line showed the section name twice (once between the clock and bar, once in the chips strip below). Dropped it from the `#nowlabel` string — it now reads `playing 0:03 · bar 3`, section still in the chips line.
10. **Removed the 📱 emoji** from the Background button label.
