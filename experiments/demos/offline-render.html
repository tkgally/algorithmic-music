OfflineAudioContext render & measure

The dev-time validation rig for the algorithmic-music project's audio layer (wiki/computational-music-metrics.md, wiki/engine-architecture.md testability requirement #2). It renders faster than real time in an OfflineAudioContext, then measures the buffer with the project's own analysis library. The libraries load as plain <script src> (file://-safe). Headless driver: experiments/tools/render-measure.mjs.
